## ğŸ“‹ CLI Command Reference (Complete)

### Chat Operations
```bash
./chat-llm.js                      # Interactive mode
./chat-llm.js "question"           # Single query
./chat-llm.js file.txt             # Batch processing
./chat-llm.js --demo               # Demo mode
./chat-llm.js --help               # Show help
```

### Sentiment Analysis (DOCUMENTED)
```bash
./chat-llm.js sentiment "text"     # Analyze sentiment
```

### Request Logging (DOCUMENTED)
```bash
./chat-llm.js stats                # View statistics
./chat-llm.js stats --type chat    # Filter by type
./chat-llm.js search "keyword"     # Search logs
./chat-llm.js export json          # Export JSON
./chat-llm.js export csv           # Export CSV
./chat-llm.js clear-logs           # Clear old logs
```

### Configuration (DOCUMENTED)
```bash
./chat-llm.js config-get path      # Get value
./chat-llm.js config-set p v       # Set value
./chat-llm.js config-list          # List all
./chat-llm.js config-save name     # Save profile
./chat-llm.js config-load name     # Load profile
./chat-llm.js config-list-profiles # List profiles
./chat-llm.js config-reset         # Reset defaults
```

### Performance Monitoring (DOCUMENTED)
```bash
./chat-llm.js perf-report          # Formatted report
./chat-llm.js perf-metrics         # JSON metrics
./chat-llm.js cache-stats          # Cache info
./chat-llm.js cache-clear          # Clear cache
./chat-llm.js export perf-json f   # Export metrics
./chat-llm.js export perf-csv f    # Export CSV
```
